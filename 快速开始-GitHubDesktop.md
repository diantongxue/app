# 🚀 快速开始 - 使用 GitHub Desktop

## 📋 3 步完成编译

### 步骤 1：在 GitHub Desktop 中添加项目（1 分钟）

1. **打开 GitHub Desktop**
2. **File** > **Add Local Repository**
3. **点击 "Choose..."**，选择：`/Users/mac/Desktop/故乡食品/app`
4. **点击 "Add Repository"**

---

### 步骤 2：推送代码到 GitHub（2 分钟）

1. **如果还没有 GitHub 仓库**：
   - 点击 **"Publish repository"**
   - 输入仓库名称
   - **选择 Public**（重要！）
   - 点击 **"Publish Repository"**

2. **提交并推送**：
   - 勾选所有文件（左侧）
   - 填写提交信息：`准备编译 Android APK`
   - 点击 **"Commit to main"**
   - 点击 **"Push origin"**

---

### 步骤 3：触发编译并下载 APK（10 分钟）

1. **打开浏览器**，访问你的 GitHub 仓库

2. **点击 "Actions" 标签页**

3. **点击 "Run workflow"** > **"Run workflow"**

4. **等待编译完成**（约 5-10 分钟）

5. **下载 APK**：
   - 点击已完成的工作流
   - 滚动到底部
   - 点击 **"app-debug-apk"** 下载

---

## ✅ 完成！

就这么简单！不需要任何本地配置！
